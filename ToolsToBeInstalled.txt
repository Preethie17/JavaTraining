Tools required for this training course:

Laptop: Must be 4gb RAM and above with latest configuration

1. Jdk 1.8 - Environment to execute the Java programs
url:https://www.oracle.com/technetwork/java/javase/downloads/jdk8-downloads-2133151.html
 -> you may need to sign up on Oracle.com in order to download the software
 -> please download correct version of your laptops, some time laptops are configured with 32 bit or 64 bits

2. Eclise - IDE (To write Java programs )
url:https://www.eclipse.org/downloads/packages/release/photon/r/eclipse-ide-java-ee-developers
 -> please download correct version of your laptops, some time laptops are configured with 32 bit or 64 bits


3. IntelliJ - IDE (for Java programs) - Optional
url:https://www.jetbrains.com/idea/download/#section=windows
 -> Community version (FREE One)

4. Git- Code repository for practice and results programs
url:https://git-scm.com/downloads
GUI Clients: https://git-scm.com/downloads/guis

5. Atom - Taking notes
url:https://atom.io/
